Summary of vocabulary introduced in this section

Property Description
property Provides a linguistic annotation. If the subject of this property is a lexical entry this refers to all instances of the lexical entry. If the subject is a form, then it describes the inflection of the given form. If the subject is a component it describes the morphological property of this component within a multi-word expression.
topic Indicates the topic of a lexicon or lexical entry.
context Indicates the context in which a given lexical sense is used
